Variables

What is a variable?

Variable = a container.

We can save a value in the variable and then use it later

Syntax rule for variables:

let and const creates a new variable

camelCase - best practice

PascalCase - used for one feature in js

kebab-case - doesn't work is js. Use in html and css and file names.

snake_case - used in other languages but not in js.

let, const and var = CREATING A VARIABLE

let, use it if you gonna change the value later in the code.

const is constant, you cannot change it later in the code. BEST PRACTICE IF YOU AIN'T CHANGING THE VALUE LATER.

var is the original way of using, but should NOT be used anymore. At all. But you micht see it in older codes.
